# WebFOCUS Chart.js グラデーション棒グラフ拡張機能

Chart.js を使用してグラデーション効果のある棒グラフを表示する拡張機能です。

## 主な機能

- **グラデーション棒グラフ**: Chart.js ライブラリを使用したグラデーション効果のある棒グラフ表示
- **視覚的な効果**: 放射状グラデーションによる立体的な表現
- **レスポンシブデザイン**: 表示領域に応じた自動調整

## データ要件

この拡張機能では以下のデータバケットを使用できます：
- **値** (メジャー): 表示する数値データ
  - 最小: 1
  - 最大: 1
- **ラベル** (ディメンション): X軸のカテゴリー
  - 最小: 1
  - 最大: 1

## ファイルの説明

### com.shimokado.radial_gradiate.js
グラフモジュールの本体

### properties.json
設定ファイル

### icons/icon.png
グラフ選択で表示されるアイコン

### lib/chart.js
Chart.js ライブラリ

### css/style.css
スタイルシート

### test.html（テスト用ファイル、デプロイ不要）
ローカルでモジュールをテストするためのHTMLファイル。WebFOCUSにデプロイする必要はない。

### tdgchart-min-for-test.js（テスト用ファイル、デプロイ不要）
ローカルでモジュールをテストするためにtdgchart-min.jsを加工したjsファイル。WebFOCUSにデプロイする必要はない。
