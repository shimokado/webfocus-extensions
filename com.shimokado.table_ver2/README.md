# WebFOCUS 横幅を圧縮した表形式（合計行付き） 拡張機能

横幅を圧縮した表形式で、合計行を含む拡張機能です。Excel出力機能も搭載しています。

## 主な機能

- **圧縮表示**: 横幅を最適化した表形式
- **合計行表示**: データの合計を表示する行を自動生成
- **Excel出力機能**: xlsx.js を使用したExcelファイル出力
- **フレキシブルなデータ表示**: 最大5つの値、ラベル、詳細をサポート
- **カスタマイズ可能なスタイル**: 
  - フォントファミリー、サイズ、色、太さを調整可能
  - 背景色、ボーダー設定
- **インタラクティブな要素**: 
  - ツールチップ機能搭載
  - センターテキスト表示オプション

## データ要件

この拡張機能では以下の3種類のデータバケットを使用できます：
- **値** (メジャー): 表示する数値データ
  - 最小: 0
  - 最大: 5
- **ラベル** (ディメンション): データのカテゴリー
  - 最小: 0
  - 最大: 5
- **詳細** (メジャー/ディメンション): 追加情報
  - 最小: 0
  - 最大: 5

## スタイル設定

- **テーブルスタイル**: 
  - フォント: Arial, sans-serif
  - サイズ: 20px
  - 色: #663300
  - 太さ: normal
  - 背景色: #f9f9f9
  - ボーダー: 1px solid #ddd
- **値ラベル**: 
  - フォント: sans-serif
  - サイズ: 自動調整
  - 色: #333333
  - 太さ: bold
- **ラベルテキスト**:
  - フォント: Verdana
  - サイズ: 14px
  - 色: #333333
  - 太さ: bold

## 外部ライブラリ

- **xlsx.js**: Excelファイル出力用ライブラリ (CDN経由で読み込み)
