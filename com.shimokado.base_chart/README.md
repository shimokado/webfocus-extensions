# WebFOCUS シンプル棒グラフ拡張機能

基本的な機能を備えたシンプルな棒グラフを表示する拡張機能です。

## 主な機能

- **シンプルな棒グラフ表示**: 構造を理解するための最小限の設定
- **D3.jsベース**: 高度な視覚化ライブラリを活用

## データ要件

この拡張機能には以下のデータバケットが必要です：
- **値** (メジャー): グラフ化する数値データ
  - 最小: 1
  - 最大: 1
- **ラベル** (ディメンション): 棒グラフのカテゴリー
  - 最小: 1
  - 最大: 1

## 設定オプション

- チャートの上部余白: 50px
- 外部ライブラリ: D3.js
