# WebFOCUS Chart.js 円グラフ拡張機能

Chart.js を使用してシンプルな円グラフを表示する拡張機能です。ラベルが複数指定された場合は、各ラベルレベルごとに別々の円グラフを表示します。

## 主な機能

- **Chart.js 円グラフ**: Chart.js ライブラリを使用した円グラフ表示
- **複数ラベル対応**: ラベルが複数ある場合、各レベルごとに円グラフを作成して表示
- **カスタマイズ可能なフォント設定**:
  - ラベルテキストのフォントファミリー、サイズ、色、太さ
  - 値ラベルのフォント設定
- **レスポンシブデザイン**: 表示領域に応じた自動調整

## データ要件

この拡張機能では以下のデータバケットを使用できます：

- **値** (メジャー): 表示する数値データ
  - 最小: 1
  - 最大: 1
- **ラベル** (ディメンション): 円グラフのカテゴリー
  - 最小: 1
  - 最大: 5

## 複数ラベル時の動作

- **単一ラベル**（例: "Country"）: 1つの円グラフを表示
- **複数ラベル**（例: ["Country", "Brand"]）: 各ラベルレベルごとに円グラフを表示
  - レベル0: Countryごとの円グラフ
  - レベル1: Brandごとの円グラフ（集計）

## スタイル設定

- **ラベルテキスト**:
  - フォント: Arial, sans-serif
  - サイズ: 12px
  - 色: #666666
  - 太さ: normal
- **値ラベル**:
  - フォント: Arial, sans-serif
  - サイズ: 14px
  - 色: #333333
  - 太さ: bold
