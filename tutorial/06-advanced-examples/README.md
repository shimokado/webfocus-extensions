# 第6章：高度な実装例（IBIリポジトリより）

## コンテンツ

1.  **[Sunburst Chart の分析](01-sunburst-analysis.md)**
    - `com.ibi.sunburst` を題材に、D3.jsの統合、モジュール分割、プレビュー表示の実装方法を解説します。

2.  **[メタ拡張機能パターン](02-meta-extension-pattern.md)**
    - `com.ibi.tutorial` で採用されている、複数の拡張機能を動的に切り替える高度なアーキテクチャについて解説します。
