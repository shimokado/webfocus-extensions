# WebFOCUS拡張グラフ開発 完全チュートリアル

## 📚 チュートリアル概要

このチュートリアルは、WebFOCUS拡張グラフ開発の基礎から実践まで、体系的に学習できるように設計されています。JavaScript、HTML、CSSの基本知識があることを前提としています。

## 🎯 学習目標

- WebFOCUS拡張機能の仕組みを理解する
- 実際に動作する拡張グラフを作成できるようになる
- デバッグ手法とベストプラクティスを習得する
- プロダクション環境へのデプロイができるようになる

## 🛠️ 開発ツール活用

### 本プロジェクトの便利ツール

#### com.shimokado.paramsの活用
```javascript
// データ構造の確認
console.log('data:', renderConfig.data);
console.log('dataBuckets:', renderConfig.dataBuckets);
```

#### NPMスクリプトの活用
```bash
# 新しい拡張機能の作成
npm run create-extension

# 拡張機能のデプロイ
npm run deploy
```

#### ブラウザ開発者ツール
- Consoleタブ：ログ出力とエラー確認
- Elementsタブ：DOM構造とCSS確認
- Networkタブ：リソース読み込み確認

### デバッグのベストプラクティス

1. **段階的な確認**
   ```javascript
   console.log('Step 1: Data received', data);
   console.log('Step 2: Data normalized', normalizedData);
   console.log('Step 3: DOM created');
   ```

2. **エラーハンドリング**
   ```javascript
   try {
       // メイン処理
   } catch (error) {
       console.error('Extension Error:', error);
       // フォールバック処理
   }
   ```

3. **データ検証**
   ```javascript
   if (!data || data.length === 0) {
       console.warn('No data available');
       return;
   }
   ```
